# AI Cybersecurity Analysis Platform

## Overview
This project implements an AI-powered cybersecurity system covering:
- Application Security
- Digital Forensics
- Threat Intelligence

## Features
- Vulnerability detection in source code
- Anomaly detection using Isolation Forest
- Malicious URL detection using Random Forest
- GUI for Threat Intelligence

## Technologies
- Python
- Scikit-learn
- NumPy
- Colorama
- Tkinter

## How to Run

### Application Security
python app_security.py

### Digital Forensics
python forensics.py

### Threat Intelligence
python threat_intel.py

### GUI
python threat_gui.py
